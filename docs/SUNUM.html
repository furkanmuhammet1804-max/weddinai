Güvenlik & Devreye Alma

Güvenlik varsayılan etkin

  • Şifreleme: hassas içerik AES-256-GCM (anahtar yalnız env).
  • API: her uçta Zod + rate-limit + yetki kontrolü.
  • KVKK: onaysız foto AI'ya gönderilmez.
  • Minimum veri: AI'ya telefon/e-posta gitmez; loglar PII-siz.
  • Dosya: private bucket + imzalı/süreli URL.
  • Sır sızıntısı: client bundle'da anahtar/kripto yok (doğrulandı).

Devreye alma

1. Env: APP_ENCRYPTION_KEY, GEMINI_API_KEY, SUPABASE_SERVICE_ROLE_KEY, NEXT_PUBLIC_SUPABASE_*

2. Migration: 0012 → 0013 → 0014 (rollback betikleri hazır)

3. next build → EXIT 0
